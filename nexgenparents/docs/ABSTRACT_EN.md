# *NexGen Parents: A Mobile Decision-Support Platform for Family-Oriented Video Game Literacy*

NexGen Parents is developed in response to a growing social and educational challenge: many families struggle to interpret gaming terminology, age classifications, and platform-specific parental control settings in a rapidly evolving digital entertainment ecosystem. The project aims to provide parents with a practical, reliable, and technically accessible tool that improves informed decision-making regarding children’s video game use. The development combines a Flutter-based cross-platform client, Firebase services for authentication and collaborative data persistence, and external API integration through RAWG to retrieve structured game metadata and discovery filters. Functionally, the system includes a moderated collaborative glossary, recommendation and filtering mechanisms by age and platform, and step-by-step parental control guides for major gaming ecosystems. From an engineering perspective, the implementation prioritizes asynchronous service consumption, fault-tolerant API handling, and local persistence of user preferences for offline continuity. The resulting product demonstrates that a focused, family-centered interface can bridge technical complexity and everyday parental needs, while also providing a scalable foundation for future educational partnerships and data-driven personalization features.

Keywords: parental controls, video game literacy, mobile application, API integration, Firebase
